# Repository Structure

## This docs repository

- `index.mdx` is the home page
- `overview` pages explain what the platform is
- `architecture` pages explain how systems fit together
- `marketplace` pages explain commerce behavior
- `backend` pages explain service ownership
- `database` pages explain persistence
- `infrastructure` pages explain runtime support
- `operations` pages explain how to work on the project
- `reference` pages hold living reference material
- `placeholders` pages mark systems that still need their own inventory imported here

## Product repositories

- `reboundjerseys` owns the current platform
- `ai-jersey-scanner` owns image analysis
- `rebound-jersey-authentication` owns authenticity evaluation
