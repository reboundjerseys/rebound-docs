# Backend Overview

The backend is the platform control plane.
It owns shared marketplace state, integration orchestration, and operational endpoints.

## Responsibilities

- authentication and trust workflows
- marketplace search and sync
- eBay integration
- warehouse intake
- AI Jersey Scanner orchestration
- Authentication service orchestration
- moderation and reporting
- Google Merchant sync
- Bria image operations
- Discord notifications
- Cybership push flows
