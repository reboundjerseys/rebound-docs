# Backend Integrations

The backend is the integration hub for the product.

## Verified integrations

- Supabase
- Stripe
- Algolia
- OpenAI
- Pinecone
- eBay
- Discord
- Google Merchant
- Cybership
- EasyPost
- Make
- Notion
- Bria
- eBay is documented here as an internal-view-only integration surface.

OpenAI powers the scanner service.
Pinecone powers the authentication service.
