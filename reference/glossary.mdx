# Glossary

## Listing

The sellable marketplace offer.

## Product

The broader jersey record that can back one or more listings.

## Order

The committed purchase state.

## Checkout draft

The pre-commit state before an order is placed.

## Conversation

The messaging thread between marketplace participants.

## Warehouse intake

The workflow that captures source inventory and turns it into platform data.

## Scan result

The structured output from the jersey analysis pipeline.

## Authentication verdict

The output of the authenticity prototype.

## Team marketplace

The team-specific storefront experience.
