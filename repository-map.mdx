# Repository Map

## `reboundjerseys`

This is the primary application repository.
It owns the web app, mobile app, backend API, Supabase schema, edge functions, and most platform integrations.

Source anchors:

- `src/backend/server/server.ts`
- `src/web/app`
- `src/frontend/app`
- `src/backend/supabase`

## `ai-jersey-scanner`

This repository owns jersey image analysis, prompt orchestration, structured extraction, and listing generation.

This documentation will be completed once the inventory for the corresponding repository has been generated.

## `rebound-jersey-authentication`

This repository owns the experimental authenticity pipeline.
It compares jersey images against reference data and returns a verdict.

This documentation will be completed once the inventory for the corresponding repository has been generated.

## Ownership boundary

- `reboundjerseys` owns the user-facing product and platform APIs.
- `ai-jersey-scanner` owns jersey identification and listing enrichment.
- `rebound-jersey-authentication` owns authenticity evaluation.
