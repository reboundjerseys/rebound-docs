# Image Upload Lifecycle

The platform has more than one image path.
The main ones are marketplace uploads, scanner uploads, and warehouse intake photos.

## Common lifecycle

```mermaid
flowchart LR
  Upload[Upload image] --> Storage[(Supabase Storage)]
  Storage --> Process[Processing step]
  Process --> AI[AI analysis]
  AI --> Product[Product or listing data]
  Product --> Search[Search index]
```

## Where each path lands

- Marketplace images support listings and profile content.
- Scanner images support attribute extraction and listing creation.
- Warehouse intake images support intake automation.
- Authentication images support reference matching.
