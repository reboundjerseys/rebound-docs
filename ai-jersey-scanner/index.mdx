# AI Jersey Scanner

The scanner is a separate repository and service.
It turns jersey images into structured attributes, model guesses, color palettes, listing copy, and insight payloads.

The platform backend sends images to it, then uses the outputs in marketplace, moderation, and warehouse flows.

## What it owns

- jersey attribute extraction
- jersey model identification
- multi-jersey detection
- title and description generation
- color palette extraction
- listing, comment, user, brand, and model insight extraction
- PostHog-backed LLM observability
